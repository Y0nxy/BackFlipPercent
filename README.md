# BackFlip%

Set the percentage on succeeding a backflip.

# How To Use
* using [ModMenu](https://thunderstore.io/c/peak/p/PEAKModding/ModConfig/)
* using a Mod Manager for Ex.  [Gale](https://github.com/Kesomannen/gale)
* Editing the .cfg file yourself

# Bonus features
* Adds a Changeable keybind for a backflip
* Can toggle off
